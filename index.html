Actividad 2 (Valor 10 puntos)

Identifica y describe las propiedades de los siguientes elementos CSS:

Tema 1-Texto

Propiedad Valor Inicial Descripción
Color Black Define el color del texto. El valor inicial es black (negro).
Font-family serif Especifica la familia de fuentes. El valor inicial común es serif.
Font-size 16px Establece el tamaño del texto. El valor inicial es 16px en la mayoría de los navegadores.
Font-weight 400 normal Define el grosor del texto. normal es equivalente a 400.
font-style Normal Establece el estilo del texto como normal, italic o oblique.
font-variant Normal Controla la variante de texto, como las versalitas (small-caps).
font normal normal 400 16px/normal serif Propiedad abreviada para definir varias propiedades de fuente. Combina estilo, peso, tamaño y familia tipográfica.
text-align start Alinea el texto dentro de su contenedor. El valor predeterminado es start (depende de la dirección del texto, como left en texto en inglés).
line-height Normal Define la altura de línea como normal, afectando el espacio vertical entre las líneas de texto.
text-decoration None Controla las decoraciones del texto como subrayado, tachado o sobrelínea.
text-transform None Controla la capitalización del texto. Los valores comunes son uppercase, lowercase, y capitalize.
vertical-align baseline Alinea elementos en línea en relación con la línea base del texto.
text-indent 0 Define la sangría de la primera línea de un bloque de texto.
letter-spacing Normal Controla el espacio entre caracteres.
word-spacing Normal Define el espacio entre palabras.
white-space Normal Controla el manejo de espacios en blanco. Puede ser normal, nowrap, pre, pre-wrap, o pre-line.

Tema 2-Enlaces

Característica Propiedades Descripción
Tamano, color y decoracion font-size, color, text-decoration Controlan el tamaño del texto del enlace, el color y la decoración (subrayado, sobrelínea, etc.).
font-size: 16px; Define el tamaño del texto del enlace. Valor inicial es 16px.
color: blue; Define el color del enlace. Valor inicial es blue (azul).
text-decoration: underline; Establece la decoración. Valor inicial es underline (subrayado).
font-family: inherit; Especifica la familia de fuentes para el enlace. Por defecto, hereda el valor de su elemento padre.
Decoración Personalizada :hover { color: red; } Cambia el color del enlace cuando se pasa el cursor sobre él (hover).
:visited { color: purple; } Define el color del enlace después de haber sido visitado (visited).
:focus { outline: none; } Elimina el contorno por defecto cuando el enlace está enfocado (focus).

Tema 3-Imagenes

Característica Propiedades Descripción
Altura y anchura width Define el ancho de la imagen. Valor inicial: auto (tamaño original de la imagen).
height Define la altura de la imagen. Valor inicial: auto (tamaño original de la imagen).
Eliminar borde border: none; Elimina el borde alrededor de una imagen con enlace. Valor inicial: none.

Tema 4-Listas

Propiedad Valor Inicial Descripción
list-style-type disc Establece el tipo de marcador para las listas. El valor predeterminado es disc.
list-style-position outside Define si los marcadores están dentro o fuera del contenido de la lista. El valor inicial es outside.
list-style-image none Permite usar una imagen como marcador en lugar de los predeterminados. El valor inicial es none.
list-style disc outside none Propiedad abreviada que permite definir list-style-type, list-style-position y list-style-image en una sola declaración.

Tema 5-Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table separate Define si las celdas de la tabla tienen bordes separados (separate) o colapsados (collapse).
border-spacing table 0 Define el espacio entre las celdas cuando border-collapse está en separate. El valor inicial es 0.
empty-cells table show Controla si las celdas vacías de la tabla se muestran o se ocultan. El valor inicial es show.
caption-side table Top Establece la posición del título de la tabla (caption). El valor inicial es top.
